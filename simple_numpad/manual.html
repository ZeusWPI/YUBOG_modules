Simple numpad :)

The simple numpad is a simple green numpad with a display showing a red number or two with a reset button at the bottom that will reset the module

  • If the displayed number is 1: press 2.
  • If the displayed number is 2: press 3.
  • If the displayed number is 3: press 4.
  • If the displayed number is 4: press 5.
  • If the displayed number is 5: press 6.
  • If the displayed number is 6: press 7.
  • If the displayed number is 7: press 8.
  • If the displayed number is 8: press 9.
  • If the displayed number is 9: press 1.
  • If the sum of all displayed numbers including previous ones lies between 10 and 99: press 4.
  • If the sum of all displayed numbers including previous ones lies between 100 and 199: press the display.
  • If the sum of all displayed numbers including previous ones lies between 200 and 299: press 1.
  • If the sum of all displayed numbers including previous ones lies between 300 and 699: press the display.
  • If the sum of all displayed numbers including previous ones lies between 700 and 799: press all numbers on the numpad in order.
  • If the sum of all displayed numbers including previous ones lies between 800 and 899: press 3.
  • If the sum of all displayed numbers including previous ones lies between 900 and 999: press 2.
  • If the sum of all displayed numbers including previous ones is greater than 1000: press reset! ( and start over again :) )
