# <ins>The Button</ins>

A button that simply says “press me” can still get you into trouble. Read the instructions and follow them exactly — do the first applicable step from the list below.

## Rules

- If the button is blue and the button says "Submit" hold down the button and follow the instructions in section **"The strip"**.
- If there is more than 1 battery on the bomb and the button says "Launch", press and immediately release the button.
- If the button is white and the serial number contains a vowel, hold down the button and follow the instructions in section **"The strip"**.
- If there are more than 2 batteries on the bomb and there are 6 or more modules, press and immediately release the
  button.
- If the button is yellow, hold the down button and follow the instructions in section **"The strip"**.
- If the button is red and the button says "Hold", press and immediately release the button.
- If none of the above apply, hold down the button and follow the instructions in section **"The strip"**.

## The strip

When holding down the button, the colored strip is lit up. Depending on the color, a different instruction needs to be followed. 

- <ins>Blue strip</ins>: release when the timer contains a 4.
- <ins>White strip</ins>: if less than half of the modules are solved, release when the timer contains an 8, otherwise release when the timer contains a 1.
- <ins>Yellow strip</ins>: release when the timer contains a 5.
- <ins>Purple strip</ins>: if the ports contain a Stereo RCA or Serial port, release when the timer contains a 2, otherwise release when the timer contains a 5.
- <ins>Any other color</ins>: release when the timer contains a 2.
